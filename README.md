Site oficial da comunidade CJC Minecraft, com informacoes sobre servidores, regras e como entrar.

- 2 servidores online
- Mais de 400 mods disponiveis
- Uptime 24/7
- Hospedagem em conteineres refrigerados da Audicom Telecom

## Regras rapidas

- Conta original do Minecraft obrigatoria.
- Hacks, cheats e exploits sao proibidos.
- PVP permitido, com respeito.
- Envie seu nickname para liberar o acesso.

## Como entrar

1. Baixe o modpack desejado pelos links acima.
2. Instale no seu launcher (CurseForge, ATLauncher, etc.).
3. Envie seu nickname para liberacao.
4. Entre pelo IP do servidor.

## Rodar localmente

Abra o arquivo index.html no navegador.
